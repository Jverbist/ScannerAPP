How to Use the Logistics Sorting Application

  1. Upload Packing Slip: Go to the Home page and upload your Excel file containing packing slip data. Ensure the file is in `.xlsx` format.
  2. Input Serial Numbers: Navigate to the Input Items page and enter item serial numbers. Use a scanner or manual entry, with each number on a new line.
  3. View Results: After submitting the serial numbers, view the matched and unmatched items grouped by purchase order.
  4. Repeat or Start Over: Return to the Home page to upload a new file or repeat the process with additional serial numbers.
