How to Use the Logistics Sorting Application

Follow this guide to understand how to use the application effectively.

  1. Step 1: Go to the Home page and upload an Excel file containing your packing slip data. Ensure the file is in `.xlsx` format and contains the necessary columns:
    • PO Number
    • Item
    • Serial Number
  2. Step 2: Navigate to the Input Items page. Enter serial numbers manually or via a barcode scanner. Make sure each serial number is on a new line.
  3. Step 3: View the sorted results grouped by POBE. The application will also display unmatched items in a separate section.
  4. Step 4: Repeat the process if needed. You can upload a new file or input additional serial numbers.
